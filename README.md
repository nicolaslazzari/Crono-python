# Crono-python
Un projet pour chronométrer les petites courses
A project to time the little races

Une application en python pour permettre l'organisation de petite courses.

À partir d'un fichier excel ou libreoffice calc bien formatter l'application récupère les inscrits et crée les courses
associées aux informations présente dans les fichier.

Ensuite après avoir lancer la course, il suffit d'appuyer sur les boutons associers aux dossards.

Pour l'exportation des résultats, l'application crée un fichier excel avec le classement général et le classement par
catégories.

Un manuel est disponible avec les instructions pour plus de details

Point Positif :
- Fait tout, de la récupération des inscrits à l'exportation des résultats

Point Négatif : 
 - N'affiche pas le temps en temps réél

Je reste ouvert à toute critique, l'application étant encore jeune
